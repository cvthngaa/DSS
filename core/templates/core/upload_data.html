⚠️ Hướng Dẫn Định Dạng File CSV

  • File bắt buộc phải có ít nhất 3 cột: name (tên), latitude (vĩ độ), longitude (kinh độ).
  • Có thể dùng tên cột viết tắt như: ten, lat, lng.
  • Dòng dữ liệu đầu tiên sẽ mặc định được tính làm Kho (Depot). Các dòng dưới là khách hàng.

Ví dụ file chuẩn:

name latitude longitude
Kho chính 10.7769 106.7009
Khách hàng 1 10.7626 106.6601
Khách hàng 2 10.7812 106.6823
